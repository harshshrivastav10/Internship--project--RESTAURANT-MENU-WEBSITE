# 🍽️ Restaurant Menu Website

A modern and responsive Restaurant Menu Website built using **HTML**, **CSS**, and **JavaScript**. The project provides an attractive interface for browsing food items by category, viewing prices, and exploring featured dishes.

## Features

* ✨ Responsive design for mobile, tablet, and desktop
* Interactive food categories
* Beautiful and modern UI 💯
* Easy navigation
* Fast-loading and lightweight
* Clean and well-structured code

## Technologies Used 🔍

* HTML5 
* CSS3
* JavaScript

## Purpose 📌

This project was created to improve front-end web development skills by building a real-world restaurant menu interface with a focus on responsive design and user experience.
